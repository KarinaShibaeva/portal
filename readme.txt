Возможности пользователя:

На стартовой странице расположен логотип сайта, название, лозунг, кнопка "Каталог кружков".
При нажатии на кнопку "Каталог кружков" осуществляется перенос на соответсвующую страницу.

На странице "Каталог кружков" расположены карточки с кружками и категории. При нажатии на название категории осуществляется переход на страницу с кружками данной категории.
При нажатии кнопки "Подробнее" в карточке кружка выполняется переход на страницу с полным описанием кружка, кнопкой "Записаться на кружок" и отзывами. На данной странице авторизованный пользователь может оставить отзыв.
При нажатии на кнопку "Записаться на кружок" пользователя перенаплавляет на страницу с формой отправки заявки.
Авторизованный пользователь может увидеть все свои отправленные заявки в личном кабинете.

Для того чтобы зарегистрироватся пользователь переходит на страницу регистрации через меню. На данной странице заполняет форму регистрации.
После успешной регистрации пользователь переходит на страницу авторизации и выполняет вход в аккаунт. Пользователя перенаправляет в его личный кабинет.

Перейдя на вкладку "Расписание" пользователь может ознакомиться с расписанием работы кружков.

Перейдя на вкладку "Наши учителя" пользователь может узнать какой учитель ведет тот или иной кружок.


Возможности админа:

Панель админа доступна под логином admin и паролем admin11

В админ панели есть возможности редактирования статуса заявок, расписания и статуса отзывов.
